# Proyecto Angular - Recuperatorio Desarrollo Web

## Descripción
Aplicación Angular para el recuperatorio de la materia Desarrollo Web que consume la API de JSONPlaceholder.

## Características
- ✅ Listado de fotos
- ✅ Agregar nuevas fotos (mediante tfoot)
- ✅ Editar fotos (con modal)
- ✅ Eliminar fotos
- ✅ Diseño responsive con tema negro/celeste

## Tecnologías
- Angular
- TypeScript
- Bootstrap
- JSONPlaceholder API

## Instalación
```bash
npm install
ng serve
